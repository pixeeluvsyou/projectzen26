Installation

Select the method for your device brand.

BOOTLOADER UNLOCKED REQUIRED. RESEARCH YOUR MODEL ONLINE.
Step 01

Requirements

Download the appropriate files for your model:

Google / Xiaomi / Others

recovery.img
boot.img
ZenOS zip package

Samsung

recovery.img (or .tar)
vbmeta.img (or .tar)
ZenOS zip package

Step 02

Flash Recovery

Standard (Fastboot)

fastboot flash recovery recovery.img
fastboot reboot recovery

Samsung (Odin)

Put device in Download Mode. Use Odin to flash:

• AP slot: recovery.tar

• CP/User slot: vbmeta.tar

Reboot to recovery immediately.

Step 03

Clean Install

In ZenOS recovery:

1. Select Factory Reset

2. Select Format Data and confirm

Step 04

Sideload ZenOS

On device: Advanced → Apply Update → Apply from ADB

adb sideload ZenOS-Neon-16-OFFICIAL.zip
Step 05

Reboot

Optional: sideload addons (GApps/Magisk) then select:

Reboot System Now
